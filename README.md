# 臺灣證券交易

## 股票資訊(正面)
   ### 收盤價 vs. 月均價：
       紅色：收盤價 > 月均價
       綠色：收盤價 < 月均價
   ### 漲跌價差：
       紅字：正成長（漲）
       綠字：負成長（跌）
   ### 漲跌停板強化：
       漲停 ( 價差 >= 9.7% )：卡片邊框顯示紅色
       跌停 ( 價差 <= -9.7% )：卡片邊框顯示綠色
   ### 可加入「個人庫存」或「我的最愛」

## 股票資訊(背面)
   取代 Alert Dialog 改由點選股票資訊卡片翻轉後顯示關鍵財務指標
   ### 本益比 (P/E Ratio)
   ### 殖利率 (%)
   ### 股價淨值比 (P/B Ratio)

## 股票篩選
   Bottom Sheet 選單：點擊右上角篩選圖示彈出導覽列
   ### 自訂排序：支援「個人庫存」或「我的最愛」
   ### 排序邏輯：支援「股票代號降序」、「股票代號升序」及「ETF」
   ### 產業/題材分類：同步載入 assets/stock_categories.json，支援自定義題材（如 AI、海運）一鍵過濾

## 股票搜尋
   ### 即時搜尋 (Real-time Search)：使用者每輸入一個字元，清單會立即過濾，無需點擊搜尋按鈕。
   ### 多欄位匹配：支援同時透過 「股票代號」 (如 2330) 或 「股票名稱」 (如 台積電) 進行檢索。

## 如何擴充產業分類？
   在 app/src/main/assets/stock_categories.json 中按此格式編輯：
  ```
  {
     "AI 概念股": ["2330", "2382", "3231"],
     "您的新分類": ["代號1", "代號2"]
  }

  ```
