# 臺灣證券交易

## 股票資訊(正面)
* **收盤價 vs. 月均價顏色邏輯**
    * 紅色：收盤價 > 月均價
    * 綠色：收盤價 < 月均價
* **漲跌價差顏色邏輯**
    * 紅字：正成長（漲）
    * 綠字：負成長（跌）
* **漲跌停板視覺強化**
    * 漲停 (價差 >= 9.7%)：卡片邊框顯示紅色
    * 跌停 (價差 <= -9.7%)：卡片邊框顯示綠色
* **個人化追蹤**
    * 支援加入「個人庫存」或「我的最愛」

## 股票資訊(背面)
* **交互設計**
    * 取代 Alert Dialog，點選卡片翻轉後顯示財務指標
* **關鍵財務指標**
    * 本益比 (P/E Ratio)
    * 殖利率 (%)
    * 股價淨值比 (P/B Ratio)

## 股票篩選
* **介面設計**
    * 點擊右上角篩選圖示彈出 Bottom Sheet 選單
* **自訂過濾**
    * 支援僅顯示「個人庫存」或「我的最愛」
* **排序邏輯**
    * 支援「股票代號降序」、「股票代號升序」及「ETF」
* **產業/題材分類**
    * 同步載入 assets/stock_categories.json
    * 支援自定義題材（如 AI、海運）一鍵過濾

## 股票搜尋
* **即時搜尋 (Real-time Search)**
    * 輸入字元時清單立即過濾，無需額外點擊
* **多欄位匹配**
    * 支援透過「股票代號」或「股票名稱」進行檢索

## 如何擴充產業分類
* **檔案路徑**：app/src/main/assets/stock_categories.json
* **JSON 格式**：
```json
{
  "AI 概念股": ["2330", "2382", "3231"],
  "您的新分類": ["代號1", "代號2"]
}
